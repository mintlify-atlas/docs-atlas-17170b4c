  <Steps>
    <Step title="Install dependencies">
      Install Bun and clone the repository to get started.
      
      ```bash
      bun install
      ```
    </Step>
    
    <Step title="Start PostgreSQL">
      Launch PostgreSQL with logical replication enabled using Docker.
      
      ```bash
      docker-compose up -d
      ```
      
      <Note>
        The Docker configuration automatically enables logical replication and creates the required publication.
      </Note>
    </Step>
    
    <Step title="Run the application">
      Start the Bun server and open your browser to see live database changes.
      
      ```bash
      bun run dev
      ```
      
      Navigate to `http://localhost:3000` to see the monitoring interface.
    </Step>
    
    <Step title="Test with real data">
      Execute SQL commands to see changes appear in real-time.
      
      ```sql
      INSERT INTO todos (title) VALUES ('Test task');
      UPDATE todos SET done = true WHERE id = 1;
      DELETE FROM todos WHERE id = 2;
      ```
      
      Watch as every operation appears instantly in your browser.
    </Step>
  </Steps>
